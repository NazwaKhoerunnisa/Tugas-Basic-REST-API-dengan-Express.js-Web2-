# Tugas-Basic-REST-API-dengan-Express.js-Web2-

## 📌 Catatan Tambahan

- Project ini telah di-push ke repository dan siap digunakan.
- Pastikan menjalankan `npm install` lalu `node app.js` untuk mulai server.

## 🖼️ Tip Screenshot (hasil Thunder Client)

- Saat mengambil screenshot endpoint REST API (contoh: `GET /students`, `POST /students`, `PUT /students/:id`, `DELETE /students/:id`), sertakan:
  1. URL lengkap (misal `http://localhost:3000/students`).
  2. Method HTTP (`GET/POST/PUT/DELETE`).
  3. Status response (`200 OK`, dll).
  4. Body response (JSON) yang tampil di panel kanan.
- Gunakan mode gelap agar kontras teks lebih jelas.
- Tambahkan keterangan singkat pada setiap screenshot: apa yang diuji, dan hasil yang diharapkan.
